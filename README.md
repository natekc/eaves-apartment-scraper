# eaves-apartment-scraper

This tool is used to scrape 1B1B apartment info from the eaves Rancho Penasquitos apartments website.

## Prereqs
python3

selenium
```
python3 -m pip install selenium
```
selenium-driver
```
sudo apt-get install chromium-chromedriver
```

## Running this tool
```
./eaves-apartment-scraper
```
